# Timekeeper

A solo time tracker built with React + PocketBase.

## Quick start

```bash
chmod +x start.sh
./start.sh
```

This will:
1. Download PocketBase if needed
2. Start the PocketBase server on port 8090
3. Create the admin account (krismcfarlin@gmail.com)
4. Start the React frontend on port 5173

Open http://localhost:5173 and log in.

## PocketBase Admin

http://localhost:8090/_/ — manage collections, data, and settings.
